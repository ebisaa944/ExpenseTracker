API Documentation

REST API endpoints for ExpenseTracker

All API endpoints require authentication. Use your regular login credentials.
Categories API
  • GET /api/categories/
    List all categories
  • POST /api/categories/
    Create a new category
  • GET /api/categories/{id}/
    Get specific category
Expenses API
  • GET /api/expenses/
    List all expenses
  • POST /api/expenses/
    Create new expense
  • GET /api/expenses/summary/
    Get expense summary
Budgets API
  • GET /api/budgets/
    List all budgets
  • GET /api/budgets/progress/
    Get budget progress
Goals API
  • GET /api/goals/
    List all financial goals
  • POST /api/goals/{id}/update_progress/
    Update goal progress
Testing the API

You can test the API using tools like:

  • Postman - Desktop API testing tool
  • Thunder Client - VS Code extension
  • curl - Command line tool
